VLC


Qu'est ce que VLC ?

VLC est un lecteur et un framework de vidéo et d'audio de code ouvert compatible avec un grand numéro de format de dossier multimedia.


Le but de se projet était de nous faire découvrir ce logiciel et a nous apprendre a l'utilisé. Lors de ce projet j'ai pu apprendre a appliqué des filtre vidéo tel que le gaussianblur, qui est un flou.

Nous avons aussi pu apprendre a ciblé une zone en particulier sur l'image d'une vidéo grace au pixel.

Nous avons utilisé VLC sur une VM (virtual Machine) sous debian 10.

Jeux

Jeu Roguelike


Ce projet était un sujet de code camps, c'est a dire que plusieurs groupe travaillait sur un meme sujet et le meilleur projet gagne a la fin du code camps.

Le but de ce projet était de nous faire découvrir l'univers du jeu vidéo. Le projet était simple, il fallait créer un jeu de type roguelike et avoir certaine fonctionnalité.

Les fonctionnalité les plus importantes étaient les mouvements du personnages, avoir une animation d'attaque et l'action d'attaque aussi. Il fallait aussi avoir au moins 5 niveaux et avoir un HUD

Lors de ce projet, nous avons réussi a répondre a toutes les demandes du sujet.


Pour la réalisation de ce projet nous avons utilisé le logiciel GODOT et on a utilisé du GDScript comme langage.

Toutes les parties du projet ont été fait par nous même, les sprites du personnage et des ennemies aussi.
